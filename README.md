# ProjekatVerovatnocaStatisika-NBA-igraca
Opis projekta

Ovaj projekat istražuje statističke podatke NBA igrača i koristi principe verovatnoće i statistike za analizu njihovih performansi. Projekat omogućava pregled ključnih statistika, kao što su poeni, asistencije, skokovi i drugi relevantni podaci, kako bi se izvukli uvidi o igračima i njihovim učincima na utakmicama.

Funkcionalnosti

Analiza individualnih statistika NBA igrača

Upoređivanje igrača po različitim kriterijumima (poeni, asistencije, skokovi…)

Prikaz osnovnih statističkih parametara i verovatnoća

Vizualni prikaz podataka kroz tabele i grafikone (ako je uključen)

Svrha projekta

Cilj projekta je da se kroz praktičnu primenu statistike i verovatnoće pokaže kako matematički principi mogu pomoći u razumevanju performansi sportista i donošenju odluka zasnovanih na podacima.
